# Live Practice

This folder contains exercises edited by Sat and reviewed with Codex.

## Workflow

1. Open the current `.ps1` exercise.
2. Edit and save it locally.
3. Ask Codex to review the file.
4. Run the script in PowerShell 7.
5. Share any output or exact errors with Codex.
6. Commit the completed exercise when you can explain every line.

## Exercises

- `01-variables.ps1` - Variables and string interpolation
- `02-numbers.ps1` - Numbers and calculations
- `03-user-input.ps1` - Interactive input with Read-Host
- `04-conditions.ps1` - Decisions with if, else and comparison operators
- `05-multiple-conditions.ps1` - Three-way decisions with if, elseif and else
- `06-arrays-and-loops.ps1` - Collections with arrays and foreach loops
