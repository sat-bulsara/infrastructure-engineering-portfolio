# Microsoft Infrastructure Engineering Portfolio

> A hands-on portfolio documenting my journey from IT Support Engineer to Microsoft Infrastructure Engineer through real-world projects, automation, troubleshooting and documentation.

---

# About

Welcome to my Infrastructure Engineering portfolio.

This repository is the central hub for my learning journey as I build the skills required to become a modern Microsoft Infrastructure Engineer.

Rather than studying theory alone, every topic is learned by designing, building, breaking, troubleshooting, documenting and automating real infrastructure.

Everything in this portfolio is built using industry tools and follows engineering best practices.

---

# Learning Philosophy

Every project follows the same engineering workflow:

1. 📚 Learn
2. 🛠️ Build
3. 💻 CLI
4. ⚡ PowerShell
5. ☁️ Azure CLI
6. 🌍 Terraform (where applicable)
7. 🐍 Python (where applicable)
8. 💥 Break
9. 🔍 Troubleshoot
10. 📝 Document
11. 🚀 Publish to GitHub

This ensures every technology is learned through practical implementation rather than memorisation.
